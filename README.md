# Logic_Digital_Discrete_Tool
 一个有关本科数字电路与离散数学的程序

基于electron^8.1.1开发

## Function

1.化简逻辑表达式

2.生成最多4元素的卡诺图

3.生成真值表以及对应的主析取/合取范式



## Install



```
cd Logic_Digital_Discrete_Tool
git clone https://github.com/lemonoil/Logic_Digital_Discrete_Tool.git
npm install
```



## Usage



```
npm start
```



## APP

```
npm run package
```
